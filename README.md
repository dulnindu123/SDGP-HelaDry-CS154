# HelaDry - Solar-Powered IoT Hybrid Dehydrator
HelaDry is an innovative academic project developed for the SDGP (Software Development Group Project) module. Our goal is to address the significant issue of post-harvest loss in Sri Lanka by providing a sustainable, tech-driven solution for food preservation.

This system utilizes a hybrid approach, combining solar energy with IoT-enabled monitoring to ensure optimal drying conditions for various agricultural products, thereby extending shelf life and increasing economic value for local farmers.

🚀 Key Features
Hybrid Power: Primary solar energy utilization with backup power systems.

IoT Monitoring: Real-time tracking of temperature and humidity levels inside the dehydrator.

Automated Control: Smart logic to maintain the ideal environment for different food types.

Sustainability: Focus on reducing food waste specifically within the Sri Lankan agricultural context.

🛠 Tech Stack
Software: Java / Object-Oriented Programming (OOP).

Design: UML for system architecture and logic schemas.

Hardware: IoT sensors (Temperature, Humidity) and solar power integration.
