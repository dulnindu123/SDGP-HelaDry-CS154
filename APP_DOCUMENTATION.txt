1. APPLICATION OVERVIEW
================================================================================

HelaDry is a mobile companion app for a solar-powered food dehydrator machine.
It allows farmers and users in Sri Lanka to:

  - Monitor their dehydrator machine in real-time (temperature, humidity, etc.)
  - Control the machine (fan speed, heater, emergency stop)
  - Track drying batches from start to finish
  - Access a crop-specific drying guide with preparation steps
  - View historical drying records
  - Connect to the machine via WiFi (Online) or Bluetooth BLE (Offline)

The app supports both LIGHT and DARK mode themes.

---------- 4.1 Manual Controls Page ----------

Allows direct control of the dehydrator machine:

  a) FAN SPEED CONTROL
     - Slider: 0% to 100%
     - Current value displayed
     - "Apply" button sends command
     - Success snackbar: "Fan speed set to X%"

  b) TARGET TEMPERATURE CONTROL
     - Slider: 30В°C to 80В°C
     - "Apply" button sends command
     - Success snackbar: "Target temp set to XВ°C"

  c) HEATER TOGGLE
     - Switch: ON / OFF
     - Immediate effect

  d) EMERGENCY STOP BUTTON
     - Red button at the bottom
     - Shows confirmation dialog
     - Sets fan to 0%, heater OFF
     - Critical safety feature


---------- 4.2 Start New Batch Page ----------

Create a new drying session:

  - Crop selector: Choose from 7 supported crops
  - Tray count: 1-10 trays
  - Weight input: Enter weight in kg
  - Auto-fills recommended temperature from crop guide
  - Starts the drying batch and tracks it

5. STATE MANAGEMENT (SessionStore)
================================================================================

The app uses Provider + ChangeNotifier for state management.
SessionStore holds ALL app state in memory (no persistent storage):

  AUTH STATE:
    - isLoggedIn (bool)
    - userName (String)
    - userEmail (String)

  CONNECTION STATE:
    - connectionMode ('online' or 'offline')
    - pairedDeviceId (String)
    - pairedDeviceName (String)
    - selectedWifiSsid (String)
    - wifiConfigured (bool)
    - savedNetworks (List)

  DEVICE CONTROL STATE:
    - fanSpeed (0-100%)
    - heaterOn (bool)
    - targetTemp (30-80В°C)

  SETTINGS STATE:
    - useCelsius (bool)
    - tempOffset (double)
    - humidityOffset (double)
    - overTempAlert (bool)
    - lowBatteryAlert (bool)
    - sensorFaultAlert (bool)

  KEY METHODS:
    - login(name, email)    вҶ’ sets logged in, stores user info
    - logout()              вҶ’ resets ALL state to defaults
    - setConnectionMode()   вҶ’ online or offline
    - setPairedDevice()     вҶ’ store device ID and name
    - setFanSpeed()         вҶ’ 0-100%
    - setHeaterOn()         вҶ’ bool
    - setTargetTemp()       вҶ’ 30-80В°C
    - emergencyStop()       вҶ’ fan=0, heater=OFF


================================================================================
6. THEME SYSTEM
================================================================================

The app supports Light and Dark themes via ThemeController.

  ThemeController:
    - Extends ChangeNotifier
    - Stores ThemeMode (light/dark/system)
    - toggleTheme() switches between light and dark

  AppTheme (app_theme.dart - 499 lines):
    - Defines AppTheme.lightTheme and AppTheme.darkTheme
    - Custom color schemes for both modes
    - Dark mode uses deep blues/teals (#0A1628, #112240, #00D4AA)
    - Light mode uses clean whites with blue accents (#1976D2)
    - Custom text themes, card themes, input themes, button themes
    - Font: Plus Jakarta Sans (Google Font)

  ModeToggleButton widget:
    - Sun/Moon icon toggle
    - Available on login, signup, and connection mode pages

10. IoT HARDWARE FIRMWARE (ESP32)
================================================================================

  Location:   SDGP-HelaDry-CS154/src/main.cpp
  Board:      ESP32 DOIT DevKit V1
  Framework:  Arduino (PlatformIO)
  Libraries:  HX711 (load cell), Firebase ESP32 Client

  Current Firmware Functions:
    1. Connects to WiFi using hardcoded SSID/password
    2. Initializes HX711 load cell sensor on pins 18, 19
    3. Calibrates scale with factor -7050.0
    4. Connects to Firebase Realtime Database
    5. Reads weight every 3 seconds
    6. Pushes weight to Firebase path: /HelaDry/current_weight

  Firebase URL: heladry-iot-24a53-default-rtdb.asia-southeast1.firebasedatabase.app
